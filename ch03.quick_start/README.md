# 快速入門

## 環境準備

- OS：macOS，Linux，Windows
- 運行環境：建議 Node LTS，最低 8.x

## 快速初始化

推薦使用腳手架：

```shell
$ npm i egg-init -g
$ egg-init egg-example --type=simple
$ cd egg-example
$ npm i
```

啟動：

```shell
$ npm run dev
$ open localhost:7001
```
